# Экспорты AI-сессий (spec §8.1)

Сюда помещать экспорты AI-сессий (markdown, jsonl или текст) — столько, сколько уместно для понимания процесса создания и эволюции кода.

**Примеры файлов в этом репо:**
- `session-export-sample.md` — публичный API (seller name/rating, default sort) и smoke-тесты.
- `session-export-admin-media.md` — Admin API и загрузка изображений в MinIO.

**Имена файлов:** `session-YYYY-MM-DD.md`, `session-*.jsonl`, или осмысленные имена вроде `session-export-*.md`.
